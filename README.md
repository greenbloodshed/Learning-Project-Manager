# Learning Project Manager

A desktop learning project manager / progress tracker built w/ Python and Tkinter.

## Purpose

This application is being built as a learning project to practice Python GUI development, object-oriented design, state management, and incremental software development.

## Current Features

- Create and View Projects
- Place these Projects in either an Active status, or a Hold status
- Select a Project to view details
- Delete Projects with confirmation

## Planned Features

- Menu bar/Right-Click dialog
- JSON save/load persistence
- Project editing
- Tracking time spent on Projects
- Better project detail view
- Improved layout and stying

## Project Structure

main.py
modules/
    __init__.py
    project.py

## How to Run

From the project root:
python main.py

## Notes

This is an active learning project. Some features may still be experimental.
